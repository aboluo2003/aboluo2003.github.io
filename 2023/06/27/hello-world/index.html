想用 github 搭一个个人 blog 已经很久了，但一直咕咕咕咕。感觉自己博客园的主题丑的一批，只有背景好看。偶然一天空闲，找了点 hexo 教程终于把 blog 搭出来了。

感觉 hexo-reimu 主题配色挺好看的，粉粉嫩嫩的，少女心泛滥了hh。因为不是东方厨所以把灵梦的 banner 换了。虽然但是，有一说一东方曲是真的好听。

这个 hello-world 直接用 hexo 的 hello-world 太寒碜了，虽然也算是个 hello-world。于是把原来的内容删了，随便写点。感觉这里就当 TODO List 和博客美化日志好了。
